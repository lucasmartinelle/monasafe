# Politique de sécurité

## Versions supportées

| Version | Supportée          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Signaler une vulnérabilité

**Veuillez ne pas signaler les vulnérabilités de sécurité via les issues GitHub, les discussions ou les pull requests publiques.**

Merci de les signaler de manière confidentielle en envoyant un email à **lucasmar.martinelle@gmail.com**.

### Informations à fournir

Veuillez inclure autant d'informations que possible afin de nous aider à mieux comprendre et résoudre le problème :

- Une description de la vulnérabilité
- Les étapes pour reproduire le problème
- La ou les version(s) concernée(s)
- L'impact de la vulnérabilité
- Les éventuelles mesures correctives que vous avez identifiées

### Ce à quoi vous attendre

- Vous recevrez un accusé de réception dans un délai de **48 heures**.
- Nous travaillerons avec vous pour comprendre et valider le problème.
- Nous vous tiendrons informé de l'avancement de la correction.
- Une fois le problème résolu, nous mentionnerons publiquement votre contribution (sauf si vous préférez rester anonyme).

## Langues acceptées

Nous acceptons les signalements en **français** ou en **anglais**.
